## 关于 EngTree

**EngTree 英语树** 是一款专为大学生和英语学习者打造的智能英语学习平台。"EngTree" 取意"夏天的树"—— 学英语不是苦役，而是像树木生长一样，自然、持续、不急不躁。每天浇灌一点，日久便枝繁叶茂。

平台品牌图标是一棵茂盛的绿树，树干旁靠着一个正在阅读的小人 —— 象征着学习者与知识之树的亲密关系。

---

## 六大核心功能

### 1. 智能任务
基于你的学习目标（四六级 / 考研 / 雅思 / 托福 / 留学 / 日常提升）和当前水平，AI 自动生成每日学习任务，涵盖词汇、听力、阅读、口语、写作五大维度。打开就能学，不需要自己规划。

### 2. 单词计划
内置 30+ 核心词汇库，采用艾宾浩斯遗忘曲线算法，在你即将遗忘的临界点精准推送复习。每张单词卡片支持 3D 翻转 —— 正面看单词，翻转看释义、音标和例句。记得更牢，忘得更慢。

### 3. 听说训练
面试、旅行、课堂、社交、日常 —— 5 个高频场景与 AI 实时对话练习。AI 以 SSE 流式输出回复，就像在和真人聊天，练出开口自信。

### 4. 作文批改
提交你的英语作文，AI 从内容、结构、语法、词汇四个维度独立评分并给出详细反馈和修改建议，比单纯的红笔批改更有方向感。

### 5. 英语树藏宝阁
55 篇精选资料库，涵盖阅读材料、写作练习、阅读理解三大类型：

| 分类 | 内容 |
|------|------|
| 外刊精选 | 社交媒体极化、无限工作周、炎症与抑郁、SpaceX 神话、复仇的代价、父职演变、美洲狮避险 |
| 名著阅读 | 傲慢与偏见、尤利西斯、白鲸、杀死一只知更鸟、长眠不醒 |
| 语法词汇 | 英语语法基础全览、美丽与魅力词汇辨析 |
| 学习方法 | 英语高效学习全攻略、学好英语没有捷径、5 部美剧学英语、5 本经典名著学英语 |

每篇资料支持收藏、点赞、阅读记录，可按分类 / 难度筛选和搜索。

### 6. 成长报告
- **雷达图**：听说读写能力分布一目了然
- **柱状图**：周学习时长追踪
- **折线图**：分数趋势变化

让努力有迹可循，不再盲目。

## 其他功能

- **新人引导**：3 步完成 —— 基本信息 → 8 个学习目标选择 → 4 级水平评估
- **错题本**：自动收录薄弱点，针对性复习不留死角

- **邮箱注册登录**：数据云端保存，多设备同步

## 设计理念

EngTree 的视觉灵感来自夏日林荫——午后透过树叶的光斑，微风拂过的绿色树冠，清新的空气和安静的蝉鸣。

| 设计要素 | 说明 |
|----------|------|
| 主色 | Summer Green（夏日绿，生机、清新、自然感） |
| 辅助色 | Warm Amber（暖琥珀，用于进度 / 得分 / 高亮） |
| 背景 | 微暖浅绿白，不是冷白，带一丝绿意 |
| 英文字体 | Source Serif 4（衬线体，学术感 + 自然感） |
| 中文字体 | Noto Sans SC（思源黑体） |

侧边栏底部和首页横幅各有一个夏日诗句 / 英语名言轮换组件，每 12 秒淡入淡出切换，涵盖夏日古诗词和语言学习名言。

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/Cecilia11-cc/engtree.git
cd engtree

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入你的 Supabase 和 AI API 密钥

# 启动开发服务器
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 即可看到应用。
